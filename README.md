## Prerequisites

- yarn v1.17.3
- node v12.3.1

### Build

#### For Local Development

#### For Chromium Development Build

```
yarn
yarn dev:chromium
```

#### For Firefox Development Build

```
yarn
yarn dev:firefox
```

#### For Production Release

#### For Chromium Build

```
yarn
yarn build:chromium
```

#### For Firefox Build

```
yarn
yarn build:firefox
```
